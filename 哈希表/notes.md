# 哈希表

常见的三种Hash结构：

1. 数组
2. Set
3. Map

最好了解一下Java中的各种容器

**当我们遇到了要快速判断一个元素是否出现在集合里的时候，就要考虑哈希法。**

哈希法是牺牲了空间换取了时间，因为我们需要用额外的容器来存放数据，才能实现快速的查找。

**如果在做题时遇到需要判断一个元素是否出现过的场景也应该第一时间想到哈希法。**

## HappyNumber

只有三种情况会出现：

1. 变到最后sum为1
2. sum在一个环里面循环，会出现重复数字
3. sum一直变越变越大，无限循环下去

第三种情况被证明是不可能的
